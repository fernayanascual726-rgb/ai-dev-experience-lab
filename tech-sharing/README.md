# Tech Sharing

本目录用于整理 AI 开发者体验相关的技术分享、内部交流材料和演讲草稿。

重点是把实践经验转化为可交流、可复盘的内容。

## 适合收录的内容

- 技术分享大纲
- 内部培训材料
- 主题演讲草稿
- 案例集合和讨论问题
- 分享后的反馈记录

## 推荐写法

每份分享材料建议说明：

1. 分享主题
2. 目标听众
3. 核心问题
4. 内容结构
5. 案例和材料来源
6. 讨论问题
7. 反馈记录

## 命名建议

```text
ai-coding-practice-sharing.md
ai-workflow-team-session.md
developer-experience-talk-outline.md
```

## 维护原则

- 分享内容应来自实践沉淀
- 保留材料来源，方便追溯
- 区分大纲、草稿和最终版本
