# Experience Summary

本目录用于沉淀 AI 开发者体验、效率变化和协作体验方面的阶段性总结。

重点关注长期观察和复盘，而不是单次案例。

## 适合收录的内容

- AI Coding 对开发流程的影响总结
- 开发者效率、心智负担和协作方式的变化
- 团队引入 AI 工具后的观察
- 阶段性实践复盘和问题清单

## 推荐写法

每篇总结建议说明：

1. 观察周期
2. 背景和样本
3. 主要发现
4. 具体证据
5. 限制和偏差
6. 后续改进方向

## 命名建议

```text
2026-q1-ai-coding-summary.md
developer-experience-notes.md
team-ai-adoption-review.md
```

## 维护原则

- 总结应基于案例、数据或持续观察
- 区分事实、判断和假设
- 记录负面体验和不确定性
