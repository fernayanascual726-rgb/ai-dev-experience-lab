# Series 4 · Workflow State

AI Workflow 的核心正在发生变化。

大多数讨论仍然停留在“生成内容”：模型是否更强、Prompt 是否更好、输出是否更准确。但真实工程系统已经进入长链路、多步骤、多 Agent、持久上下文、状态恢复和 Workflow 编排阶段。

真正决定 AI 系统可靠性的，已经不只是模型能力，而是状态管理能力。

## 核心主题

- AI Workflow State
- Context Engineering
- Agent State
- Durable Workflow
- Persistent Memory
- Workflow Recovery
