# Templates

本目录存放统一 Markdown 模板，用于保持内容结构、标签和写作风格一致。

## 使用方式

1. 根据内容类型复制对应模板
2. 补全文档元信息和正文
3. 使用标签体系中的标准标签
4. 发布前检查是否保留了背景、约束和限制

## 模板列表

| 模板 | 用途 |
| --- | --- |
| `article-template.md` | 通用文章、案例、总结和分享材料 |
| `prompt-template.md` | 开发提示词记录和复盘 |
| `workflow-template.md` | AI 工作流设计与复盘 |

## 维护原则

- 模板用于统一结构，不限制具体表达
- 必填项应服务于长期复用
- 不追求完整包装，优先保留真实上下文
