# 适用场景

说明这个 prompt 适合的任务、输入条件和不适用边界。

# Prompt

```text
请基于以下上下文完成任务。

目标：

背景：

相关文件或材料：

约束条件：

请先说明你的理解，再给出处理步骤。不要扩大范围，不要修改未提到的内容。
```

# 使用建议

- 先补全真实背景，再要求 AI 输出
- 明确输入和输出边界
- 对复杂任务先让 AI 分析，不要直接生成最终结果
- 保留人工 Review 和验证步骤

# 风险

- 背景不足时容易得到泛化回答
- 约束不清时容易出现过度改动
- 输出可能缺少可验证依据

# 适合谁

- 希望把 AI 纳入日常开发流程的开发者
- 需要沉淀团队 prompt 规范的工程团队
- 正在复盘 AI Coding 效果的人
