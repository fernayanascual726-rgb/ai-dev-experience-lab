# GitHub 项目介绍

## 简短描述

面向 AI Developer Experience 的长期知识库，记录 AI Coding、AI 工作流、提示词实践和开发者效率复盘。

## About 建议

A long-term knowledge base for AI Developer Experience, covering AI Coding practices, AI workflows, prompt patterns, collaboration notes, and developer productivity reflections.

## 中文介绍

`ai-dev-experience-lab` 是一个面向 AI 开发者体验的长期知识库。

它用于记录 AI 在真实研发流程中的落地方式，包括 AI Coding 场景案例、开发提示词、AI 工作流、人机协作方式、开发者效率和团队体验复盘。

这个仓库不追求工具展示或观点包装，更关注实践上下文、适用边界和可复用的方法。

## 推荐 Topics

- `ai-coding`
- `developer-experience`
- `ai-workflow`
- `prompt-engineering`
- `ai-collaboration`
- `dev-productivity`

## 适用场景

- 记录个人或团队 AI Coding 实践
- 沉淀 AI 工作流和协作规范
- 复盘开发者体验与效率变化
- 准备技术分享和团队交流材料
