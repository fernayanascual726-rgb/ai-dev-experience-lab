```text

```

## 使用步骤

1. 补全背景信息
2. 明确任务边界
3. 让 AI 先分析再修改
4. 人工检查关键输出

## 输出检查点

- 是否引用了真实代码或上下文
- 是否说明了假设
- 是否覆盖约束条件
- 是否存在过度改动
