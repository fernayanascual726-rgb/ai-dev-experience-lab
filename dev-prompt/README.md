# Dev Prompt

本目录用于沉淀面向开发任务的 AI 提示词模板、写法和复盘。

重点不是收集大量 prompt，而是记录在真实开发任务中稳定有效的表达结构和约束方式。

## 适合收录的内容

- 代码理解、重构、测试、排查问题等开发提示词
- 面向 Claude Code、GitHub Copilot、ChatGPT 等工具的提示词模板
- Prompt 迭代前后的效果对比
- 针对团队协作的提示词规范

## 推荐写法

每个提示词模板建议说明：

1. 适用场景
2. 输入信息
3. Prompt 模板
4. 使用步骤
5. 输出检查点
6. 常见问题

## 命名建议

```text
code-review-prompt.md
bug-investigation-prompt.md
refactor-planning-prompt.md
```

## 维护原则

- Prompt 应该说明适用范围
- 模板优先保留结构，而不是绑定某个具体任务
- 记录失败模式和人工检查点
